About

A tracing for Akka cluster deployments made easy. Library includes Akka extension which allow traces to be collected and sent to Scribe collector and viewed using Zipkin. You can add custom annotations and key-value pairs to traces and use them for filtering via Zipkin Web UI. Extension can easily handle sampling thousands of traces per second.
